# tavily-best-practices

Build production-ready Tavily integrations with best practices baked in. Reference documentation for implementing web search, content extraction, crawling, and research in agentic workflows, RAG systems, or autonomous agents.

## What You Can Build

### AI Chatbot with Real-Time Search

Build a chatbot that can answer questions about current events and up-to-date information.

**Example prompts:**
- "Build a chatbot that integrates Tavily search to answer questions with up-to-date web information"
- "Add Tavily search to my internal company chatbot so it can answer questions about our competitors"

### News Dashboard with Sentiment Analysis

Create a live news dashboard that tracks topics and analyzes sentiment.

**Example prompts:**
- "Build a website that refreshes daily with Tesla news and gives a sentiment score on each article"
- "Create a news monitoring dashboard that tracks AI industry news and sends daily Slack summaries"

### Lead Enrichment Tool

Build tools that automatically enrich leads with company data from the web.

**Example prompts:**
- "Build a lead enrichment tool that uses Tavily to find company information from their website"
- "Create a script that takes a list of company URLs and extracts key business information"

### Competitive Intelligence Agent

Build an autonomous agent that monitors competitors and surfaces insights.

**Example prompts:**
- "Build a market research tool that crawls competitor documentation and pricing pages"
- "Create an agent that monitors competitor product launches and generates weekly reports"

## Installation

Install the Tavily CLI and best practices skill:

```bash
curl -fsSL https://cli.tavily.com/install.sh | bash
npx skills add tavily-ai/skills --skill tavily-best-practices
```

Then restart your agent to load the skill.

## Usage

Use the skill in your AI agent by invoking `/tavily-best-p`
