# Isaac Sim ROS 2 Bridge

> Source: `isaac-sim/IsaacSim/skills/isaac-sim-ros2-bridge/SKILL.md`
> https://github.com/isaac-sim/IsaacSim/blob/main/skills/isaac-sim-ros2-bridge/SKILL.md

## Overview

ROS 2 integration for Isaac Sim using OmniGraph. Covers:

- **ROS 2 bridge nodes** — `isaacsim.ros2.bridge.*` namespace
- **Nav2 integration** — full Nav2 stack in Isaac Sim
- **Multi-robot namespacing** — running multiple robots with distinct namespaces
- **Prerequisites** — ROS 2 setup, DDS configuration

## When to Use This Skill

- Integrating Isaac Sim with a real or simulated ROS 2 system
- Setting up Nav2 navigation for a mobile robot
- Running multiple robots with ROS 2 in a shared simulation
- Debugging ROS 2 bridge connectivity issues

For the complete ROS 2 bridge setup, Nav2 configuration, and multi-robot namespacing, refer to the source file above.
