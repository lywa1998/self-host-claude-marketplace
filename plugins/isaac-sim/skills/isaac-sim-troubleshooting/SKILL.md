# Isaac Sim Troubleshooting

> Source: `isaac-sim/IsaacSim/skills/isaac-sim-troubleshooting/SKILL.md`
> https://github.com/isaac-sim/IsaacSim/blob/main/skills/isaac-sim-troubleshooting/SKILL.md

## Overview

Reference for common Isaac Sim Kit 110 issues and their resolutions:

- **Startup hangs** — python_server, extension loading, asset root
- **MDL freezes** — material rendering issues
- **Physics stepping hangs** — solver deadlocks, CCD failures
- **Replicator hangs** — sensor capture blocking
- **Hydra issues** — rendering backend failures

## When to Use This Skill

- Diagnosing a hanging or frozen Isaac Sim instance
- Investigating performance issues
- Troubleshooting physics simulation problems
- Debugging Replicator or Hydra issues

For the complete issue catalog and resolution steps, refer to the source file above.
