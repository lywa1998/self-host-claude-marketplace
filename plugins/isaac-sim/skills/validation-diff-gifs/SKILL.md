## Overview

Generate animated GIFs showing pixel-level differences between validation captures and golden reference images. Provides fast visual triage of rendering benchmarks.

## When to Use This Skill

- Comparing benchmark renders against golden data
- Visually debugging rendering pipeline issues
- Identifying lighting or material regressions
- Quick regression testing of visual output

For the complete GIF generation pipeline and diff analysis, refer to the source file above.
