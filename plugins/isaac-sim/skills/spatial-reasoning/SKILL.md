## Overview

Transform mathematics and spatial reasoning for robotics simulations. Covers:

- **mpu** — matrix-plus-vector transformations
- **Bounding boxes** — AABB/OBB calculations
- **T·R·S ordering** — translation, rotation, scale composition
- **Look-at** — camera/robot target orientation
- **Collision-free grids** — placement algorithms
- **A* path planning** — grid-based search
- **Dubins paths** — curvature-constrained optimal paths
- **R-tree** — spatial indexing
- **2D bin packing** — placement optimization
- **Standards** — OSHA/NFPA/RMI safety standards

## When to Use This Skill

- Computing robot footprints and clearances
- Designing placement algorithms for warehouse items
- Calculating optimal paths with curvature constraints
- Spatial indexing for large environments

For the complete transform math patterns and safety standards reference, refer to the source file above.
