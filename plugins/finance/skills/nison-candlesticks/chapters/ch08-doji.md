# 第八章 神奇的十字线

## Core Idea
十字线（开市价=收市价）是极其重要的反转信号，尤其在上升趋势中。

## Frameworks Introduced
- **北方十字线**：上升趋势中的十字线，有效顶部反转信号
- **南方十字线**：下降趋势中的十字线，通常不作为可靠底部反转信号
- **长腿十字线（黄包车夫线）**：长上影线+长下影线，市场"失去方向感"
- **墓碑十字线**：开=收=最低价，超长上影线，看跌
- **蜻蜓十字线**：开=收=最高价，超长下影线，看涨

## Key Concepts
- 十字线在箱体区间内无预测意义
- 十字线在顶部反转时比底部反转有效得多
- 近似十字线判断：看周围变化、是否在转折点、是否超买/超卖

## Anti-patterns
- 夸大十字线意义：只表示趋势可能转变，不是立即卖出信号
- 十字线创新高收市价时应等待看跌验证

## Key Takeaways
1. 十字线显示多空力量完全平衡
2. 北方十字线比南方十字线更可靠
3. 墓碑十字线看跌，蜻蜓十字线看涨
