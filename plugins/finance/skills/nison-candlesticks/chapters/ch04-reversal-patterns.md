# 第四章 反转形态

## Core Idea
反转形态并不意味着趋势一定逆转到相反方向，而是表示之前趋势可能发生变化。应理解为"趋势变化形态"。

## Frameworks Introduced
- **伞形线（锤子线/上吊线）**：单根蜡烛线，下影线≥实体2倍，实体在价格区间上端
  - 下跌趋势后=锤子线（看涨），上涨趋势后=上吊线（看跌）
  - 上吊线需要后续的看跌验证信号
- **吞没形态（抱线）**：两根颜色相反的蜡烛线，后者实体完全覆盖前者
  - 看涨吞没：下降趋势中白实体吞没黑实体
  - 看跌吞没：上升趋势中黑实体吞没白实体
- **乌云盖顶形态**：顶部反转，黑实体收市价至少穿入前白实体50%以上
- **刺透形态**：底部反转，白实体必须穿入黑实体中点以上

## Key Concepts
- 反转形态仅表示趋势可能变化，不保证反转
- 趋势反转信号与主要趋势方向一致时才能开新头寸
- 吞没形态的高低点构成支撑/阻挡水平

## Anti-patterns
- 上吊线未经验证就做空：必须等待后续收市价低于上吊线实体
- 不看风险报偿比就交易：先考虑风险报偿比再根据形态交易

## Key Takeaways
1. 同形态在不同趋势方向上意义可能相反
2. 上吊线必须等待后续验证信号
3. 吞没形态中第二天实体越长信号越强
