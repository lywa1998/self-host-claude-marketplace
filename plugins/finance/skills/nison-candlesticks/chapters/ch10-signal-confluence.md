# 第十章 蜡烛图信号的汇聚

## Core Idea
当多个蜡烛图信号在同一区域汇聚时，该支撑或阻挡区域的重要性大大增强。

## Frameworks Introduced
- **蜡烛线簇（Candle Cluster）**：一群蜡烛线或信号在同一区域相互验证
  - 何时使用：多个独立来源的信号在同一价格区域时
  - 如何：综合蜡烛图形态、窗口、影线长度等识别关键价格区域

## Key Takeaways
1. 多个信号在同一区域时该区域的重要性倍增
2. 蜡烛线簇是联系东西方技术的核心概念
3. 信号汇聚比单个信号可靠性高得多
