# 第一章 引论

## Core Idea
蜡烛图技术源自远东，经受了数百年的市场检验。它为技术分析提供了一种全新视角：在同一张图表上同时呈现趋势方向和驱动趋势的潜藏力量。

## Frameworks Introduced
- **蜡烛图 vs 线图的比较优势**：使用同样的四类价格数据，蜡烛图能额外提供线图无法获取的市场信号
- **信息量框架**：不同理论源头的手段相互验证更有效，同源手段叠加验证意义不大

## Key Concepts
- **蜡烛线**：由实体和影线构成
- **实体**：白色(收>开)或黑色(收<开)
- **上影线/下影线**：最高价/最低价

## Key Takeaways
1. 保留传统工具同时获得独特新信号
2. 不同源头相互验证比同源手段更有价值
3. 蜡烛图不提供价格目标，需结合西方分析
