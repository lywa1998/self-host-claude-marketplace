# 第六章 其他反转形态

## Core Idea
本章讨论的形态中，孕线、平头等属于较弱反转信号；三山、三川、塔形等属于较强反转信号。

## Frameworks Introduced
- **孕线形态**：长实体后跟一个被包在其中的小实体，市场"上气不接下气"
  - 十字孕线形态：小实体为十字线时技术意义更强
- **平头顶部/平头底部**：两根蜡烛线具有相同水平的高点（顶部）或低点（底部）
  - 日线/日内图上需要其他蜡烛图信号验证；周线/月线图本身即为重要信号
- **捉腰带线**：开市价=最低价（看涨）或开市价=最高价（看跌）的长实体
- **向上跳空两只乌鸦**：罕见的看跌形态，两个小黑实体向上跳空
- **三只乌鸦**：三根依次下降的黑实体，看跌
- **白色三兵挺进（白三兵）**：三根依次上升的白实体，看涨
  - 前方受阻形态/停顿形态：白三兵的减弱版本
- **三山形态/三川形态**：分别对应西方三重顶/三重底
  - 三尊顶部形态=头肩顶，倒三尊底部=头肩底
- **圆形顶部/平底锅底部**：圆弧形+跳空确认
  - 平底锅底部需向上跳空完成，意味着市场已"对坏消息免疫"
- **塔形顶部/塔形底部**：形如高塔，两侧长实体中间小实体
- **反击线（约会线）**：两根颜色相反且收市价相同的蜡烛线，反转信号

## Key Concepts
- 数字3在日本文化中重要，许多蜡烛图形态基于3
- 一线簇概念：一群蜡烛图信号集中加强某个支撑/阻挡区域

## Key Takeaways
1. 孕线形态是小实体被长实体包在其中（与吞没形态顺序相反）
2. 三山形态与西方头肩形等价，但日本早用了100多年
3. 平底锅底部需要向上跳空才能完成（区别于普通圆形底部）
4. 塔形形态通常在行情变化后期完成，更适合长线交易者
5. 三只乌鸦形态在第三根蜡烛线完成时市场已回落较大幅度
