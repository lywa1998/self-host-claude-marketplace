# 第五章 星线

## Core Idea
星线是实体较小的蜡烛线，与前一根蜡烛线实体之间存在价格跳空，表明当前趋势驱动力正在瓦解。

## Frameworks Introduced
- **启明星形态**（三线底部反转）：长黑实体→小实体→长白实体推进到第一根实体内部
- **黄昏星形态**（三线顶部反转）：长白实体→星线→黑实体扎入第一根实体内部
- **十字启明星/十字黄昏星**：中间为十字线时信号更强
- **弃婴顶部/弃婴底部**：十字线前后均有价格跳空，极罕见
- **流星形态**：上影线长、实体在底部，出现在上升趋势后
- **倒锤子形态**：与流星同形状但出现在下降趋势后，需后续看涨验证

## Key Concepts
- **星蜡烛线**：实体小，与前一根蜡烛线实体不重叠
- **十字星线**：实体缩为十字线的星线

## Anti-patterns
- 黄昏星完成时市场已远离高点：可等待反弹至阻挡区域
- 倒锤子需要后续验证信号
- 流星形态通常不如吞没形态或黄昏星信号强

## Key Takeaways
1. 星线显示多空力量从一边倒转为平衡
2. 第三根实体深入第一根的程度决定有效性
3. 十字星线比普通星线反转意义更强
4. 弃婴形态极罕见但信号极强
