# 第九章 蜡烛图技术汇总

## Core Idea
通过完整图例分析展示各种蜡烛图形态的综合运用。图表分析带有主观性，不存在严格规则。

## Key Concepts
- 没有两个分析师会看到完全相同的图形信号
- "听说不等于经验"：只有通过亲身实践才能发掘全部潜能
- 多个信号相互验证增强支撑/阻挡区域的重要性

## Anti-patterns
- 完美锤子线下影线应为实体2倍，但实际中1.5倍是否算数取决于个人经验
- 任何形态必须放在总体技术环境中考量

## Key Takeaways
1. 各类形态应综合运用而非孤立解读
2. 蜡烛图不提供价格目标，一定要结合西方技术分析
3. 实践出真知——必须在自己关注的市场上反复验证
