# 第七章 持续形态

## Core Idea
持续形态意味着形态完成后市场仍将恢复先前的趋势。市场正在休整，然后继续原有方向。

## Frameworks Introduced
- **窗口（价格跳空）**：前一时段最高点与本时段最低点之间的价格真空地带
  - 向上窗口=看涨，构成支撑区域；向下窗口=看跌，构成阻挡区域
  - 最后一个窗口未被关闭前趋势不变
- **上升三法**：长白实体→小实体群→长白实体创新高（看涨持续）
- **下降三法**：长黑实体→小实体群→长黑实体创新低（看跌持续）
- **高价位跳空突破形态**：上冲后小实体整理→向上窗口打开
- **低价位跳空突破形态**：下跌后小实体整理→向下窗口打开
- **分手线形态**：两根颜色相反但开市价相同的蜡烛线，持续信号

## Key Concepts
- 窗口的关键支撑/阻挡：向上的窗口底部，向下的窗口顶部
- 西方"价格跳空"=日本"窗口"

## Key Takeaways
1. 窗口是最重要的持续形态，构成关键支撑或阻挡
2. 市场必须收市价突破窗口才能改变趋势方向
3. 上升三法中小实体须在第一根蜡烛线的全部价格范围内
4. 窗口之后的蜡烛线颜色无关紧要——窗口本身才是关键
